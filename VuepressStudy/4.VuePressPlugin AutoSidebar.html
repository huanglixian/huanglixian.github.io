# 介绍

这是为 vuepress 自动生成侧边栏的插件。

如果你只是写一个简单介绍页，那没必要使用它，但如果你希望用 vuepress 记录很多的东西，例如：飞跃高山与大洋的鱼的笔记 (opens new window) (opens new window)，那你可以试一试这个插件。

# 安装

npm i vuepress-plugin-auto-sidebar -D
1

# 使用

注意，请勿将 plugins 放在 themeConfig 中，如何使用插件 (opens new window) (opens new window)。

module.exports = {
  plugins: [
    ["vuepress-plugin-auto-sidebar", {
      // options
    }]
  ]
}
