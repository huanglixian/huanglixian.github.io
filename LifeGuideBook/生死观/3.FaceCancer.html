# 面对绝症，你怎办？

我是一个很怕死的人，所以我常常会想象一些极端的情形，然后思考在那种情况下我应该怎么做，好让自己有所准备，以避免那一刻到来时我会茫然无措。
例如，我常常想：

如果我发现有绝症，我应该怎么应对它？

# 我见过的做法

绝症，在身边亲友中也偶有发生，我大概见证过几种应对方式：
1、盲从医生
这应该是最常见的应对方式了。
人在面对绝症的时候，是恐惧、无助的，这时候，唯一能相信的专业人士，大概就只有医生了。
加上大多数人本身并不具备多少医学知识，他们也只能把医生当成唯一的救命稻草。
当然，如果遇到一个有水平、有医德、有策略、有耐心的医生，我们"听话照做"也许就是最好的策略。
但大多数时候，我们不容易遇到集上面四者于一身的医者（至少我们难以判断），因此，“听话”很多时候无疑是一场赌博，很难保证一定能带来好的结果。
我身边就不乏"听话但却没有好结果"的案例。
倒不是说医生故意为之，只是医生也不过是"人"罢了，我们怎么能确保每个医生都是"有水平、有医德、有策略、又耐心"的呢？
既然不能，盲从肯定是不够理性的策略。
2、瞎找偏方
另一个极端，就是不信医生，自己去找偏方，这种做法在中国大妈身上尤为常见。
-- 绿豆汤降火
-- 红豆水祛湿
-- 鱼腥草消炎
-- 蒲公英茶降血压
-- 蜈蚣酒治肿瘤
-- 板蓝根预防新冠病毒
-- 血馒头治肺痨（出自鲁迅的《药》）
......
我不敢说这些偏方一定无效（事实上大多数也是无效的，有些甚至被证明有害，比如鱼腥草中含有的"马兜铃内酰胺"会致癌）。
但如果仅凭这些偏方，而不去医院进行诊断和治疗，肯定是有问题的。
我妈可能会反驳说：

人家谁谁谁，吃了就好了。
殊不知
-- 那些吃好的，不见得是那个偏方治好的。
-- 那些吃死的，没告诉你无效罢了。
一切不能证实，也不能证伪的，都只是“传说”。
3、等死
等死，其实也是特别常见的应对方式。
但等死分两种，一是"消极等"，二是"积极等"。
消极等，就是不想去治疗（又或者不敢去治疗），但内心又极度不安。[-0.5]
这种等死，纯属逃避，大概率病情会快速恶化。
逃避，永远都不会是解决之道。
积极等，就是不愿意去治疗，但心态非常好，不去治疗只是为了最后的日子不要折腾自己，过得舒服些。[-0.5]
我的一些医生朋友跟我说过，如果真的到了绝症无法医治的时刻，他们宁愿舒舒服服地过完最后的时光，因为看过太多痛苦的医治病例了。
如果真的能摆好心态，如电影《遗愿清单》中的爱德华.科尔，在生命的最后时光从从容容地完成遗愿，也未尝不是一个好的解决方案。
只是，有多少人可以在那个时刻如此乐观、积极呢？
所以，以上三种做法，要么不可取，要么不容易，那还有没有其它更积极的思路呢？[-0.5]

# 达拉斯买家俱乐部

若干年前，我曾看过一部电影叫《达拉斯买家俱乐部》，主人翁患上了艾滋病，医生给他发了死亡通知单——他的生命只剩30天，但他却不认命，仍旧不屈不挠地寻找治疗方案。
为了活下去，他努力去搞清楚：
-- 艾滋病的底层原理
-- 目前有哪些成熟的治疗方式
-- 还有哪些未经批准的治疗方案。
最终，他找到了一些更有效但又未经批准的抗艾滋病药物，还成立了一个"达拉斯买家俱乐部"，专门走私来自世界各地未经批准的药物，除了给自己服用，还卖给其他艾滋病人。
虽然，他的行为受到了政府和药企的阻挠，最终无法持续，但他的生命因此多延长了7年，同时帮助了很多艾滋病患者。
这是一个发生在80年代的真实故事，一部西方版的《我不是药神》。
这部电影，让我第一次意识到，原来面对绝症，还能这样做。

# 雷·达里奥

另一个让我获得启发的故事，发生在一个商业红人身上——畅销书《原则》的作者雷·达里奥。
几年前的一次体检，医生发现达里奥食道里有一个病变，医生说，这个病变很大概率会导致食道癌，而这个癌症是没有办法治疗的……
基本上，医生的意思就是——绝症，建议保守治疗，尽量延长寿命。
虽然他的主治医生是约翰·霍普金斯医院的（美国最牛的医院之一）的权威专家，但达里奥不认为应该直接听从这个医生的建议，而是去请教了另一个很牛的医生。
那个很牛的医生给了达里奥一个完全不同的答案。医生说，你这个病可以治啊，但要做一个大手术，把食道和胃的一大部分切除掉，而手术的成功率是90%。
达里奥听到两个医生的观点如此不同，觉得很震惊，就干脆把他俩约到一起聊聊。
然后，他目睹了一个非常有意思的局面，两个大牛医生一个说要保守等死，一个说要做大手术。
他们并没有吵起来，两人很有职业风度地交换了看法，但最后并没有达成共识，而是保留各自的意见。
看到问题并没有解决，达里奥又找到了第三个专家。
第三个专家给的意见居然是——你这个病很好治，只要你每三个月到我这里检查一下，病变要是长出来，我用一个设备深入你的食道把病变刮掉就行了，效果跟你做大手术的是一样的。
达里奥索性又找了两个医生，这两个医生都比较同意第三个专家的看法，而且他们建议先拿个设备深入食道，做个切片检查，看看到底有没有癌变。
于是，综合上述几个专家的建议，达里奥最终选择了风险最小的方案。
结果一做切片检查，达里奥的食道里根本没有什么病变，完全是虚惊一场。

# 给自己和读者的建议

不知道看完上面两个故事，读者的感觉是怎样的。
于我而言，这两个故事颠覆了我对绝症的治疗态度，对于绝症，原来除了"对医生盲从"、"另辟蹊径"和"放弃治疗"以外，还有如此科学的解决思路。
对此，我有两点启发：
1、生命，是我们存活于世的根本，健康的体魄，是我们一切活动的基础。罹患绝症时，花足够多的时间去了解它，去搞清楚它的作用机理，是我们对于生命的基本义务。
2、那些只听取一个医生的意见行为，都是对生命的不负责任，就更不用说那些"仅仅听信偏方"或“求助于身边朋友”的行为了。
所以，如果绝症真的出现了，我会试着推自己去：
-- 找些针对性的医学书籍、课程，搞清楚病的原理。
--咨询尽可能多的医生的意见。
-- 甚至看看最新的医学研究成果（如果看不懂英文就雇个人帮忙），看看医学研究的最新进展是如何的。
读者可能会觉得，这样做很难、很麻烦吧！
但是，生命仅有一次，麻烦一下又怎么了？
